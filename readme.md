# Neon's GameMode Maker
a <a href="hitbox.io">hitbox.io</a> mod that lets you make your own custom game modes using javascript.

Original mod for <a href="bonk">bonk.io</a> was made by Sneezing Cactus, their repo being acessible <a href="https://github.com/SneezingCactus/gmmaker">here</a>
- - -
### Install guide
head over to this <a href="https://greasyfork.org/pt-BR/scripts/488480-neon-s-gamemode-maker">link</a> and with a tampermonkey extension install the script,
then you are good to go.

the GMM Button will show up whenever you are the host in custom games, which will allow you to import, export and play the custom gamemodes.
